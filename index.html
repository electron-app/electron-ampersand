Ampersand

A markdown editor.

Note: Ampersand is WIP and not close to something usable at the moment.

Ampersand is a lightweight markdown editor for Windows, Mac & Linux focused on a minimal & clean interface for distraction-free writing.

Features

  • Live Preview
  • Github Flavored Markdown
  • Dark/Light/Split Mode
  • Full screen mode
  • Emoji support
  • Export as html

Build

  1. Clone repo git clone https://github.com/jackd248/electron-ampersand.git
  2. Change to repo cd electron-ampersand
  3. npm install && npm start

Download

You can download the latest stable version of Ampersand for Windows, Mac OS and Linux.
